# Car Event MAP -- Development Documentation (MVP Stage)

## 1. Project Overview

Car Event MAP is a web application that allows users to discover car
meetups and events by location. The current MVP includes:

-   Interactive map using Leaflet
-   Spot database stored in Supabase (PostgreSQL)
-   Next.js frontend
-   Supabase database connection
-   Spot display on the map
-   Event database structure prepared

# 5. Database Tables

## areas

Search regions (Hakone, Miyagase etc.)

### SQL

``` sql
CREATE TABLE areas (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  region TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

Example

  id   name
  ---- --------
  1    箱根
  2    宮ヶ瀬
  3    大黒

------------------------------------------------------------------------

## spots

Physical locations displayed on the map.

### SQL

``` sql
CREATE TABLE spots (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  area_id BIGINT REFERENCES areas(id),
  lat DOUBLE PRECISION,
  lng DOUBLE PRECISION,
  created_at TIMESTAMP DEFAULT NOW()
);
```

Example

  id   name                         lat        lng
  ---- ---------------------------- ---------- -----------
  1    大黒PA                       35.46130   139.68009
  2    アネスト岩田スカイラウンジ   35.18424   139.04902
  3    宮ヶ瀬 鳥居原園地            35.53890   139.22234

------------------------------------------------------------------------

## events

Event information connected to spots.

### SQL

``` sql
CREATE TABLE events (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  spot_id BIGINT REFERENCES spots(id),
  event_date DATE,
  start_time TIME,
  description TEXT,
  source TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

Example

  id   name                     spot_id
  ---- ------------------------ ---------
  1    大黒ナイトミーティング   1
  2    箱根朝会                 2
  3    宮ヶ瀬ナイトオフ         3

# 7. Map Component (DB connected)

File

    components/Map.tsx

``` tsx
"use client";

import { useEffect, useState } from "react";
import { MapContainer, TileLayer, Marker, Popup } from "react-leaflet";
import { supabase } from "@/lib/supabase";
import L from "leaflet";
import "leaflet/dist/leaflet.css";

const markerIcon = new L.Icon({
  iconUrl: "/marker-icon.png",
  iconRetinaUrl: "/marker-icon-2x.png",
  shadowUrl: "/marker-shadow.png",
  iconSize: [25, 41],
  iconAnchor: [12, 41],
});

export default function Map() {

  const [spots, setSpots] = useState([]);

  useEffect(() => {

    async function loadSpots(){

      const { data } = await supabase
        .from("spots")
        .select("*");

      setSpots(data || []);
    }

    loadSpots();

  }, []);

  return (
    <MapContainer center={[35.35,139.35]} zoom={9} style={{height:"600px"}}>

      <TileLayer
        attribution="© OpenStreetMap"
        url="https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png"
      />

      {spots.map((spot:any)=>(
        <Marker key={spot.id} position={[spot.lat,spot.lng]} icon={markerIcon}>
          <Popup>{spot.name}</Popup>
        </Marker>
      ))}

    </MapContainer>
  );
}
```
