# Car Event MAP -- 開発ドキュメント（MVP段階）

## 1. プロジェクト概要

Car Event MAP は、車好きがミーティングやイベントを地図ベースで検索できる
Web アプリです。

現在の MVP で実装されている機能

-   Leaflet を使ったインタラクティブ地図
-   Supabase（PostgreSQL）に保存されたスポットデータ
-   Next.js フロントエンド
-   Supabase とのデータ接続
-   地図上へのスポット表示
-   イベントテーブルの設計

# 4. データベース設計

## ER構造

    areas
      │
      ├── spots
      │      │
      │      └── events

意味

    Area = 検索エリア
    Spot = 実際の場所
    Event = ミーティング / イベント

# 5. データベーステーブル

## areas

エリア検索用

### SQL

``` sql
CREATE TABLE areas (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  region TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

例

  id   name
  ---- --------
  1    箱根
  2    宮ヶ瀬
  3    大黒

------------------------------------------------------------------------

## spots

地図に表示されるスポット

### SQL

``` sql
CREATE TABLE spots (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  area_id BIGINT REFERENCES areas(id),
  lat DOUBLE PRECISION,
  lng DOUBLE PRECISION,
  created_at TIMESTAMP DEFAULT NOW()
);
```

例

  id   name                         lat        lng
  ---- ---------------------------- ---------- -----------
  1    大黒PA                       35.46130   139.68009
  2    アネスト岩田スカイラウンジ   35.18424   139.04902
  3    宮ヶ瀬 鳥居原園地            35.53890   139.22234

------------------------------------------------------------------------

## events

イベント情報

### SQL

``` sql
CREATE TABLE events (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  spot_id BIGINT REFERENCES spots(id),
  event_date DATE,
  start_time TIME,
  description TEXT,
  source TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

例

  id   name                     spot_id
  ---- ------------------------ ---------
  1    大黒ナイトミーティング   1
  2    箱根朝会                 2
  3    宮ヶ瀬ナイトオフ         3

# 7. Map コンポーネント（DB接続版）

ファイル

    components/Map.tsx

``` tsx
"use client";

import { useEffect, useState } from "react";
import { MapContainer, TileLayer, Marker, Popup } from "react-leaflet";
import { supabase } from "@/lib/supabase";
import L from "leaflet";
import "leaflet/dist/leaflet.css";

const markerIcon = new L.Icon({
  iconUrl: "/marker-icon.png",
  iconRetinaUrl: "/marker-icon-2x.png",
  shadowUrl: "/marker-shadow.png",
  iconSize: [25, 41],
  iconAnchor: [12, 41],
});

export default function Map() {

  const [spots, setSpots] = useState([]);

  useEffect(() => {

    async function loadSpots(){

      const { data } = await supabase
        .from("spots")
        .select("*");

      setSpots(data || []);
    }

    loadSpots();

  }, []);

  return (
    <MapContainer center={[35.35,139.35]} zoom={9} style={{height:"600px"}}>

      <TileLayer
        attribution="© OpenStreetMap"
        url="https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png"
      />

      {spots.map((spot:any)=>(
        <Marker key={spot.id} position={[spot.lat,spot.lng]} icon={markerIcon}>
          <Popup>{spot.name}</Popup>
        </Marker>
      ))}

    </MapContainer>
  );
}
```
